# Security Policy

Finance Close Control Tower is a public portfolio demo that uses synthetic data only.

Do not submit, upload, commit, paste, or share private data with this project or its public demo. This includes employer or client exports, bank records, tax records, payroll data, customer or supplier lists, credentials, API keys, tokens, webhook URLs, and personal identifiers.

The MVP public app is read-only/demo-oriented and runs from bundled synthetic sample data. It does not require secrets, external APIs, user uploads, private files, database persistence, or production integrations.

If you find a security or data-protection issue in the repository, open a GitHub issue with a minimal description and no sensitive data. If the concern involves accidental private data exposure, remove the sensitive material from any report and contact the repository owner privately before sharing details publicly.
