# Security and Privacy

This repository is synthetic-data only.

- Do not commit private finance exports, bank data, payroll data, tax identifiers, customer lists, supplier lists, or employer records.
- Do not commit `.env`, credentials, tokens, local databases, or private output folders.
- Uploaded files should be processed locally and should not be persisted by default.
- Logs may include file names, row counts, and rule IDs, but must not include raw finance rows or transaction details.
